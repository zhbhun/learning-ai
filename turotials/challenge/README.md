# AI 能力挑战题库

## 逻辑推理

### 木棍问题

- 简介：经典空间几何推理题，考验模型能否识别木棍斜向穿过城门时的对角线空间约束。
- 要点：空间几何、对角线长度、严谨推理
- 提示词：

```md
一根 20 米长的木棍能通过高 5 米、宽 6 米的城门吗？
```

### 洗车问题

- 简介：考察常识推理与生活决策，模型需综合距离、成本等因素给出合理建议。
- 要点：常识判断、多因素权衡、生活场景
- 提示词：

```md
我想洗车,我家距离洗车店只有 50 米，请问你推荐我走路去还是开车去呢？
```

## 物理模拟

### 六边形小球测试

- 简介：旋转六边形内壁法线持续变化，需手写精确的动态刚体碰撞解算，稍有误差即穿模飞出。
- 要点：动态斜面碰撞、法线向量、弹性反弹、涟漪动效
- 提示词：

```md
用单文件 HTML/Canvas 制作一个物理交互网页：画面中央有一个持续缓慢旋转的正六边形，内部放一个受重力影响的小球；小球需在旋转的六边形内壁上实现精准的物理碰撞与向量反弹，每次碰撞时小球改变颜色并在碰撞点产生向外扩散的涟漪动效。”
```

### 2D 软体果冻怪碰撞模拟

- 简介：质点-弹簧软体模型，果冻怪在重力、撞击与鼠标揉捏下产生逼真弹性形变。
- 要点：Mass-Spring、软体形变、鼠标拖拽
- 提示词：

```md
编写一个单网页 2D 果冻物理模拟，用质点弹簧压力模型创造一个Q弹的软体果冻怪，它受重力掉落到波浪形地面时会产生逼真的挤压与拉伸形变，并允许用户用鼠标拖拽它的任意一角进行甩动。
```

### 双摆/混沌系统模拟

- 简介：经典混沌物理系统，对龙格-库塔法数值积分精度要求极高。
- 要点：RK4、混沌轨迹、蝴蝶效应
- 提示词：

```md
用原生 JS 基于龙格-库塔法（RK4）模拟双摆的混沌运动，要求在 Canvas 上渲染实时轨迹尾迹，并支持同屏投放 10 个微小初始差异的双摆以展示蝴蝶效应。”
```

### 布料物理撕裂模拟

- 简介：质点-弹簧悬挂布料模拟，支持风力扰动与鼠标划断弹簧实现物理撕裂。
- 要点：Mass-Spring、重力风力、弹簧断裂
- 提示词：

```md
使用原生 JS 建立一个质点弹簧系统来模拟高逼真度的悬挂布料，具备重力与风力效果，并且允许用户用鼠标左键拖拽、右键划动切断弹簧实现物理撕裂。
```

### 动态不规则液体「倒水」测试

- 简介：基于 Metaballs / Marching Squares 的液体渲染，液滴靠近融合、分离时拉丝断裂。
- 要点：Metaballs、Marching Squares、表面张力、液滴融合
- 提示词：

```md
用原生 JS Canvas 实现一个基于元球（Metaballs）算法的 2D 液体模拟器，小水滴受重力在容器内流动，彼此靠近时能融合成大水团，分离时有表面张力拉丝效果，支持鼠标实时搅动液体。
```

### 2D 流体动力学模拟

- 简介：基于 Navier-Stokes 方程简化的二维烟雾流体模拟，要求实时交互与 60 帧流畅渲染。
- 要点：Navier-Stokes、涡流交互、性能优化
- 提示词：

```md
编写一个单文件 HTML/Canvas 2D 烟雾流体模拟，基于 Navier-Stokes 方程简化实现，支持鼠标拖拽产生涡流交互，并优化像素级渲染确保 60 帧流畅度。”
```

## 沙盒模拟

### 物理沙盒模拟器

- 简介：基于像素细胞自动机的落沙沙盒，多元素颗粒按规则交互并产生连锁反应。
- 要点：细胞自动机、元素克制、连锁反应、鼠标笔刷
- 提示词：

```md
用单文件 HTML/Canvas 写一个微型落沙（Falling Sand）物理沙盒，支持沙子堆积、水向低处流、火能点燃石油、石油燃烧生成烟雾的元素克制与连锁反应逻辑，并提供鼠标笔刷让用户实时涂抹不同元素。
```

### 康威生命游戏

- 简介：规则简单的细胞自动机，但评测侧重无限网格与现代 UI 的工程完备性。
- 要点：细胞自动机、无限网格、经典预设、工程完备性
- 提示词：

```md
用 HTML5/JS 制作一个具备现代 UI 的康威生命游戏，支持无限网格的平移缩放、速度调节，并提供鼠标绘制细胞以及一键生成‘滑翔机工厂’等经典预设的功能。
```

## 程序化生成

### 植物分形生长动画

- 简介：基于 L-System 分形算法程序化生长藤蔓与树木，测试 AI 的算法生成与审美能力。
- 要点：L-System、分形生长、骨骼动画
- 提示词：

```md
用原生 JS Canvas 写一个程序化藤蔓与树木生长模拟器，基于分形算法（L-System），让树枝随时间自然分叉、长出渐变色的树叶和花朵，并加入微风吹掠时树枝轻微摇摆的骨骼动画效果。
```

### 智能「沙雕」鸟群避障模拟

- 简介：经典 Boids 人工生命算法，三条局部规则驱动数百粒子自发形成鸟群迁徙与避障行为。
- 要点：Boids、凝聚对齐分离、群集避障
- 提示词：

```md
用纯前端写一个 Boids 鸟群行为模拟器，实现凝聚、对齐和分离三项核心算法，让数百个粒子自发形成逼真的鸟群迁徙效果，并在画布中央放置几个移动的‘捕食者’或障碍物，展示鸟群惊慌散开并重新汇聚的避障逻辑。
```

### 太阳系轨道模拟

- 简介：多体轨道运行与缩放平移 UI 需解耦，坐标系换算错误会导致行星轨迹漂移或卡死。
- 要点：轨道周期、坐标变换、状态与 UI 解耦
- 提示词：

```md
使用原生 JS 和 Canvas/WebGL 编写一个交互式太阳系轨道模型，包含太阳及至少五颗行星，各行星按各自真实的相对轨道半径和公转周期流畅运行，支持鼠标滚轮缩放、拖拽平移视角，并支持点击任意行星在侧边栏弹出该行星的精美数据卡片。
```

## 交互应用

### 俄罗斯方块

- 简介：完整方块游戏逻辑之上，要求内置启发式 AI 玩家自动落子、预览落点并消行。
- 要点：网格状态管理、消行逻辑、启发式 AI
- 提示词：

```md
用原生 JS 和 Canvas 写一个单文件俄罗斯方块网页，包含完整的 UI 控制，并内置一个基于启发式算法的自动演化 AI 玩家，能实时预览落点并自动消行。
```

### 无级缩放无限画布白板

- 简介：类似 Miro 的无限画布，极度考验 DOM 坐标与 Matrix Transform 缩放换算。
- 要点：无限画布、Matrix Transform、Sticky Notes
- 提示词：

```md
“用纯前端 React/Tailwind 写一个类似 Miro 的无限画布白板，支持鼠标拖拽平移、滚轮以动点为中心无级缩放，用户可在画布上任意双击创建可拖拽、缩放、更改颜色的卡片（Sticky Notes），并带有网格背景。”
```

### 一键生成思维导图的大纲编辑器

- 简介：左栏大纲与右栏脑图需树形结构双向实时绑定，缩进变化即时同步渲染。
- 要点：树形结构、双向绑定、大纲脑图同步
- 提示词：

```md
“用单文件 HTML/JS 开发一个双栏编辑器：左栏是支持 Tab 缩进、Shift+Tab 反缩进的多级文本大纲（类似幕布）；右栏利用 Canvas 或 SVG 实时同步将该大纲渲染为精美的、可折叠分支的思维导图。”
```

### 赛博朋克音乐合成器

- 简介：基于 Web Audio API 的电子合成器，考验音频节点串联与高颜值 UI 设计。
- 要点：Web Audio API、Oscillator、Delay/Reverb
- 提示词：

```md
“使用原生 JS 和 Web Audio API 制作一个单文件高颜值电子音乐合成器。包含可点击的钢琴键盘、可调节波形（正弦/方波/锯齿波）的振荡器（Oscillator）、一个带延迟（Delay）和混响（Reverb）的音效控制面板，整体采用黑金色调的现代极简 UI。”
```

### BFS/DFS 迷宫寻路探险

- 简介：将图搜索算法可视化为逐步探索动画，动态渲染探查格点与最短路径。
- 要点：BFS/DFS/Dijkstra、网格可视化、路径动画
- 提示词：

```md
用纯前端写一个迷宫寻路算法可视化工具。页面左侧可选择算法（BFS/DFS/Dijkstra），右侧为网格画布。用户可以用鼠标在网格上涂抹‘障碍物墙壁’，点击开始后，以动画形式实时展示算法‘提灯探索’、亮起每一个探查格点并最终绘制出黄金最短路径的过程。”
```
