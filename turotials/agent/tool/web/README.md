# 网络

联网搜索与网页信息获取。

## WebSearch

搜索网页,返回带标题和 URL 的结果块。**该工具仅返回美国地区的搜索结果。**

要点:

- 当前月份为 **2026 年 7 月**——检索近期信息时以此为准。
- 通过 `allowed_domains` 仅保留指定域名的结果,通过 `blocked_domains` 排除指定域名的结果,两者均可选。
- 完成回答后,在结尾以 **"Sources:"** 形式列出实际引用过的 URL,使用 markdown 链接。

参数:

- `query`(必填):搜索查询语句。
- `allowed_domains`(可选):仅纳入这些域名的结果。
- `blocked_domains`(可选):排除指定域名的结果。

## WebFetch

获取指定 URL 的页面内容,并按给定 prompt 处理后返回(通常输出为 markdown)。适合读取文档、文章、API 响应等网络内容,并按需提取或总结信息。

- 每次只获取一个 URL;内容较长时会被截断到适合上下文的长度。
- `prompt` 用来说明你想从页面中得到什么(如"提取所有接口的签名"或"总结核心论点")。

参数:`prompt`(必填,描述你想从该 URL 获取的信息)、`url`(必填,目标 URL)。
