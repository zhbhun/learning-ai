# [OpenSpec](https://github.com/Fission-AI/OpenSpec)

## 用法

1. `/opsx:propose <what-you-want-to-build>`

  - openspec/changes/xxxx/proposal.md
  - openspec/changes/xxxx/design.md
  - openspec/changes/xxxx/spec/xxx/spec.md
  - openspec/changes/xxxx/tasks.md

2. `/opsx:apply`
3. `/opsx:chiarchive`

  - openspec/xxx/spec.md
  - openspec/yyy/spec.md
  - openspec/zzz/spec.md
