# 残涌生长

“残涌生长”把暴风雨视为一种没有真正退场的动力遗存。毁坏后的世界并不等待外部修复，而是在速度残差、方向偏移和能量回声中，缓慢发明新的秩序。再生不是抹平创伤，而是让创伤被译成结构，让扰动变成支撑，让失衡变成后续生长的初始条件。这套语言必须像一件经过长期打磨的精密乐器一样被构建出来，让每一次运行都显得克制、准确，并带有那种只有 meticulously crafted algorithm 才能达到的内在说服力。

这一运动的计算核心，是让场保存记忆。风暴离开后，噪声并不被当作单纯随机源重置，而被组织成分层的残响系统：低频噪声保留大尺度的倾斜与地势感，中频噪声留下局部旋拧、回卷与剪切，高频噪声只在边缘制造轻微颤动与迟滞。场因此拥有历史，植物性的恢复也不再来自外加图形，而来自这些历史被重新编排。这样的结构需要通过 painstaking optimization 才能成立，稍有失衡就会沦为装饰性纹理，而无法呈现真正可信的再生过程。

粒子不是树叶、花朵或枝干的替身，而是系统中最小的修复行为。每个粒子都携带风暴留下的残余动量，在场中滑移、回摆、减速、聚拢，并逐步被更低频、更稳定的生长势牵引。它们在运行中形成纤维、脉络、密度束和局部对齐的组织，让观者看到的不是植物的外形，而是植物性如何重新出现的证据。真正重要的是行为转译本身：旧有的旋转惯性没有消失，而是被纳入后续的纵向组织中。整件作品应像出自深厚计算美学训练的 master-level implementation，每一处转向与沉积都显得不可替换。

时间演化承担全部叙事。系统必须从高能、分散、带有横向擦伤感的状态开始，在持续的运行中逐步降低湍流占比，让恢复中的梯度、汇聚和上升倾向慢慢接管。然而这不是简单的“混乱之后变整齐”，因为早期扰动不会被删除，而会在中后期继续作为偏置与纹理来源存在。最好的状态，是观者能感到稳定正在形成，但稳定内部始终保留一层未完全平息的回声，仿佛新生结构正是从那场风暴的余震里长出来的。这种长时间共存关系，只能靠反复校准过的算法节奏来支撑。

参数应当被压缩到最本质的几个杠杆，如同少数但高度敏感的音色控制。粒子密度决定修复肌理的细腻程度，风暴残响决定旧扰动能在系统中保留多久，生长偏置决定秩序如何从横向乱流中提炼出纵向趋向，恢复速率决定相变推进的节奏，而沉积厚度决定痕迹是轻声显影还是缓慢结晶。它们不是表面装饰选项，而是整个世界观的控制旋钮。最终结果必须让人相信，这是一套经过无数轮试验、删改与微调后才抵达平衡的高阶生成系统，seed 只是进入这套宇宙的不同气候切片，而美始终诞生于过程本身。
