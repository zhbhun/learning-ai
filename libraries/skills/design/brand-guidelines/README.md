# brand-guidelines

把产出物套上 Anthropic 官方品牌风格，主要是统一配色和字体层级，让内容看起来更像 Anthropic 的视觉体系。

## 使用场景

生成或润色 PPT、文档、页面文案排版、可视化稿、展示材料

## 工作原理

主要做的事有这些：

- 应用品牌主色

  - 深色 #141413
  - 浅色 #faf9f5
  - 中灰 #b0aea5
  - 浅灰 #e8e6dc

- 应用强调色
  
  - 橙色 #d97757
  - 蓝色 #6a9bcc
  - 绿色 #788c5d

- 应用字体规则

  - 标题用 Poppins，没有就降级到 Arial
  - 正文用 Lora，没有就降级到 Georgia

- 自动根据背景选更合适的文字颜色
- 非文字图形会优先使用这几个品牌强调色，保证视觉上统一又不死板
