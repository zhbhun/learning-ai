```
从前，在一个宽阔的农场里，鸭妈妈孵化了几只鸭蛋。太阳照耀在稻草堆上，小鸭子们一个接一个地破壳而出，只听见“嘎嘎嘎”的叫声此起彼伏。最后一个鸭蛋裂开时，出来的小鸭子却与众不同。它的身体笨重，羽毛灰暗，和兄弟姐妹们相比显得格外地丑陋。
“哎呀，这也是我的孩子吗？”鸭妈妈惊讶地说。
其他小鸭子看着它，窃窃私语：“看那个新来的，真是太丑了！”
丑小鸭感到很难过，但它还是尽力地跟着其他小鸭子学习游泳、觅食。然而，无论它做什么，总是被嘲笑和排斥。甚至连农场上的其他动物也开始取笑它，小猪“哼哼”着嘲笑它游泳的样子，小马也对它的走路方式发出嘲讽。
一天，丑小鸭实在无法忍受这样的生活，决定离开农场去寻找一个能被接受的地方。它踏上了旅程，经过了许多艰难困苦。冬天来临时，丑小鸭几乎冻死在冰冷的湖面上。幸运的是，一位善良的农夫发现了它，把它带回家中照顾。
春天到来时，丑小鸭的羽毛变得光滑闪亮，身形也变得优雅。它决定离开农夫的家，再次出发寻找自己的位置。当它来到一个美丽的湖泊时，看到一群高贵的白鸟在水中游弋。
“我可以加入你们吗？”丑小鸭胆怯地问。
“当然可以，欢迎你，美丽的天鹅！”一只雄壮的天鹅回答。
丑小鸭惊讶地看着自己在水中的倒影——它已经变成了一只美丽的天鹅。所有经历的磨难都让它变得更加坚强和优美。
最后，丑小鸭以天鹅的身份回到了那个农场。其他动物都惊讶地看着它变化如此巨大。那些曾经嘲笑它的小鸭子和动物们都感到羞愧。
故事告诉我们：不要因为一时的缺点就自暴自弃，尤其不要因为外表而自卑。每个人都有改变和成长的可能，重要的是要相信自己，并勇敢地面对困难与挑战。
```
